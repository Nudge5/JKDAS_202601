# Dataset and Evaluation Resources for the JKDAS 2026 Study
# JKDAS 2026研究用データセットおよび評価リソース

This repository contains the datasets and evaluation processes used in the study published in **JKDAS (Journal of the Korean Data Analysis Society) 2026**.

本リポジトリには、**JKDAS（Journal of the Korean Data Analysis Society）2026年掲載研究**で使用されたデータセットおよび評価プロセスが含まれています。

The repository includes the collected responses from both a **general-purpose LLM model** and a **RAG model trained with infectious disease information**, along with the corresponding **RAGAS evaluation scores**.

本リポジトリには、**汎用LLMモデル**および**感染症情報データで学習されたRAGモデル**の応答収集結果と、それらに対する**RAGAS評価スコア**が含まれています。

These resources are provided to support reproducibility and transparency of the experimental results presented in the study.

これらのリソースは、研究で報告された実験結果の再現性および透明性を確保するために提供されています。
